3 Exactly what is requested

Three scopes, a short list of endpoints, low volume.

The request is scoped to listing management for my own shop. Nothing about buyers, transactions, or anyone else's data.

OAuth scopes

Requested for a single seller account: mine.

R
listings_rRead my listings so sync and marketing match what is live on Etsy.
W
listings_wCreate draft listings, upload images, and update price and stock.
R
shops_rRead my own shop profile to confirm the tool is pointed at the right shop.
Not requested: transactions, billing, address, cart, favorites, feedback, or any scope touching buyers or other sellers.

Endpoints the tool calls

Etsy Open API v3. The badge shows which use case each endpoint serves.

Low volume. A short batch run when I add products, a small daily sync. Dozens of calls a day, far under the default limits.
OAuth 2.0 with PKCE. Tokens live in a local file on my computer, never in a repo, never shared.
Backoff built in. The tool respects rate limit headers and retries politely with exponential backoff.
Nothing auto publishes. The API writes drafts and sync updates. Publishing stays a manual decision.

4 Responsible use

What this tool does, and what it will never do.

It does

Boring, seller side plumbing for one shop.

  • ✓Works only on my shop. Every call is scoped to siargaopointbreak, a shop I own.
  • ✓Creates drafts for my review. Listing quality stays my responsibility, the API just saves the typing.
  • ✓Keeps Etsy accurate. Price and stock updates flow from my records to Etsy so buyers see the truth.
  • ✓Sends shoppers to Etsy. Pins and posts exist to bring buyers to the Etsy listing, not away from it.
  • ✓Follows the Etsy API Terms of Use and the trademark and branding guidelines.

It never does

Hard lines, by design.

  • ✕No scraping. No crawling of Etsy pages, search results, or other shops. The API is the only door.
  • ✕No other people's data. No buyer data, no other sellers, no marketplace analytics.
  • ✕No resale or sharing of Etsy data. Listing data is used to make my own pins and posts, nothing else.
  • ✕No automated reviews, messages, or purchases. The tool never acts as a buyer or contacts anyone.
  • ✕No public app. One seller, one machine, zero third party users. Keys stay private.
